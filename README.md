# AR Pet Studio (Next.js)

This project has been migrated from a static HTML/JS app to a Next.js app.

## Run

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Notes

- 3D assets are served from `public/models`.
- Icons and sounds are served from `public/assets`.
- Animation buttons are generated from each model's actual embedded animation list at runtime, so non-existent animations are not shown.
